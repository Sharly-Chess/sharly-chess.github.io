Version 3.6 of _Sharly Chess_ is now available!

This new version brings many user-requested features to streamline event management.

## Importing/Exporting Events

It is possible to import events from the application (Home > Create an Event > Import an SCE File), and exporting data in SCE format has been promoted to avoid the data loss encountered with other formats.

## Importing Players in CSV Format

It is possible to import players in bulk using a CSV file.

## Player Distribution in Tournaments

Before the start of an event's tournaments, it is possible to distribute players into tournaments based on their rating, either manually or automatically (Tournaments tab > Distribute Players).

This feature will be particularly appreciated by organizers of events with by-level tournaments, allowing for very quick tournament setup as soon as check-in is complete.

## Removing Unpaired Players

Previously, removing unpaired players (for example, before upload them online) was done individually on the Players tab.

This can now be done all at once (Tournaments tab > Actions > Remove Unpaired Players).

## Improvement to the Crosstable Document

Hovering over player now displays their history.

## Pairing Engine Version Update

The _bbpPairings_ engine has been updated to version 6, bringing _Sharly Chess_ into compliance with recent changes to the FIDE Handbook.
