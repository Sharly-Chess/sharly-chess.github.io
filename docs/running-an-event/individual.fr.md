# Événements individuels

Dans un **événement individuel**, les joueur·euses concourent et sont classé·es
individuellement. La mise en place commune — créer l'événement,
ajouter les tournois et inscrire les joueur·euses — est la même pour tout événement
et est couverte sous [Gérer un événement]({% link docs/running-an-event/index.fr.md %}).
Les pages ci‑dessous couvrent le travail propre aux événements individuels :

- **Le processus de pointage** — marquer les présent·es avant l'appariement.
- **Gérer les appariements** — générer les appariements, saisir les résultats, dérouler les rondes.
- **Liste des prix** — définir et attribuer les prix.

{: .note }

> :information_source: La plupart de ces étapes valent aussi pour les **événements
> par équipes**. Voir la section [Événements par équipes]({% link docs/team-tournaments/index.fr.md %})
> pour ce qui change lorsque les joueur·euses concourent en équipes.
