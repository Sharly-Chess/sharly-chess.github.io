# Individual events

In an **individual event**, players compete and are ranked individually. The common setup — creating the event, adding tournaments and registering players —
is the same for any event and is covered under [Running an Event]({% link docs/running-an-event/index.en.md %}).
The pages here cover the individual-specific work:

- **Check-in Process** — mark players present before pairing.
- **Managing Pairings** — generate pairings, enter results, run the rounds.
- **Prize List** — define and award prizes.

{: .note }

> :information_source: Most of these steps apply to **team events** too. See the
> [Team events]({% link docs/team-tournaments/index.en.md %}) section for what is
> different when players compete in teams.
