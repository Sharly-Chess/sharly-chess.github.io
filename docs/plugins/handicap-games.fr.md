Le module **Parties à handicap** est conçu pour les **tournois amicaux ou de club**, où les joueur·euses les moins fort·es bénéficient d’un **avantage de temps** face à leurs adversaires plus fort·es.

Une fois activé, trois nouveaux champs apparaissent dans la page **Paramètres du tournoi** :

- **Pénalité (secondes)** – Le nombre de secondes soustraites à la pendule du·de la joueur·euse le·la mieux classé·e.
  Cette valeur est multipliée par le nombre de *paliers Elo* séparant les deux joueur·euses.
- **Palier Elo** – La différence Elo correspondant à un palier de handicap.
  Par exemple, si la valeur est fixée à 100, une différence de 300 points donnera trois paliers de handicap.
- **Temps minimum (secondes)** – Le temps minimal attribué à chaque joueur·euse, quelle que soit la différence de classement.

Les temps ajustés pour chaque joueur·euse sont affichés dans l’écran des **appairages**.

{: .note }
> :information_source: Vous devez également définir les paramètres de cadence du tournoi.

{: .warning }
> :warning: Ce module est destiné uniquement aux tournois **amicaux ou d’entraînement** — il ne doit **pas** être utilisé pour des compétitions officielles comptant pour le classement Elo.
