Ce plug-in permet de mettre en ligne les tournois sur le site [_Chess-Results.com_](https://chess-results.com/Default.aspx?lan=20&SNode=S3).  

Il est possible de le faire manuellement depuis le menu de gauche (`Mettre en ligne` > `Chess-Results.com`). 
Une fois qu'un tournoi a été mis en ligne, vous pouvez également activer l'envoi automatique pour que toute modification du tournoi soit mise à jour en temps réel sur _Chess-Results.com_.  

Vous trouverez sur l'onglet Tournois les liens permettant d'accéder à l'interface de gestion des tournois sur le site.
