# The _FFE_ Plugin

The _SCF_ plugin provides functionality that's useful for running tournaments associated with the **Swiss Chess Federation**.

{: .note }
> :information_source: The _SCF_ plugin is automatically activated for tournaments of federation "SUI".

Enabling the plugin adds a number of a features to _Sharly Chess_:

- Downloading and local search of the player database provided by the _SCF_.
- Extra player information (player's _SCF_ code).
