# Le plug-in _SCF_

Le plug-in _SCF_ fournit des fonctionnalités utiles pour organiser des tournois rattachés à la Fédération Suisse des Échecs (_**S**wiss **C**hess **F**ederation_).

{: .note }
> :information_source: Le plug-in _SCF_ est automatiquement activé pour les tournois de la fédération « SUI ».

L’activation de ce plugin ajoute plusieurs fonctionnalités à _Sharly Chess_ :

- Téléchargement et recherche locale dans la base de données joueur·euses de la _SCF_.
- Information supplémentaire sur les joueur·euses : code _SCF_.
