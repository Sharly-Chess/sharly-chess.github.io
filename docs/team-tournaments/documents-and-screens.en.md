# Team documents & screens

Team events add a few document and screen types on top of the usual ones.

## Documents

- **Match sheets** — one table per match for the arbiter to record board results
  and collect signatures. (For Molter, where boards are paired across teams, the
  sheet uses a flat-board layout.)
- **Team ranking** — the team standings, with the tie-breaks you have configured.
- **Team Berger grid** — the round-robin encounter grid with match scores.
  Available for the **Team Round-Robin** system.

## Screens

The on-screen displays for team events are:

- **Team check-in** — teams mark themselves present for the round.
- **Team matches** — the round's pairings shown as team-vs-team blocks of boards.
- **Results entry** — per-board result input.
- **Team ranking** — the live team standings.

{: .note }

> :information_source: Federation plugins can add further team documents (such as
> competition-specific pairing sheets). See the plugin's own page.
